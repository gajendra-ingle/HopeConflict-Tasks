Biography

Linus Torvalds, born on December 28, 1969, in Helsinki, Finland, is a renowned computer programmer best known for creating the Linux kernel. His work has significantly impacted the open-source software community, and Linux has become a prominent operating system used globally.

Achievements

1991 - Created the Linux kernel and the version control system Git

2004 - Named one of the "100 most influential peopl in thw world" by Time magazine in 2004

2010 - Received an honorary doctorate from the university of Helsinki 2010

2015 - Received the 2015 IEEE Computer Society Computer Pioneer Award

2019 - Received the 2019 ACM SIGOPS Hall of Fame Award
